# Fase 2 — Endpoint, sistemas e LOGS
**Semanas 8–12 · Objetivo:** aprender a ler o que as máquinas registram. É o coração do trabalho de SOC N1.

## Checklist
- [ ] Logs do Windows: **Event Viewer**, principais Event IDs de segurança (logon 4624/4625, criação de processo, etc.)
- [ ] Logs do Linux: `/var/log`, `journalctl`, autenticação (`auth.log`)
- [ ] Conceitos de **EDR/XDR** e como diferem de antivírus tradicional
- [ ] Cisco **Endpoint Security** (parte do path gratuito)

## 🎯 Entrega da fase
Num computador seu, provocar um evento (login errado várias vezes) e achar o registro disso nos logs. Documentar com prints.
